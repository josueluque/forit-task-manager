# Task Manager

Es una aplicación web para organizar tus tareas de manera fácil y práctica, desarrollado con Express y React. La cual permite crear, editar y gestionar tareas, además de marcarlas como completadas mediante un sistema de check. 

## Clonar el proyecto
```bash
git clone https://github.com/josueluque/forit-task-manager.git
cd forit-task-manager
```

## 1. Ejecutar Backend
```bash
cd backend
pnpm install
pnpm dev
```

## 2. Ejecutar Frontend
```bash
cd frontend
pnpm install
pnpm dev
```
